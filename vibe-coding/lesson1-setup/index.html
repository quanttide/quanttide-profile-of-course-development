⏱ 2分钟

场景 2 vc-s2-deepseek-auth ⏱ 15分钟
注册 DeepSeek 账号并获取 API 密钥，为后续氛围编程开发环境搭建提供认证凭据。
📋 访问官网并注册账号
打开浏览器，访问 DeepSeek 官方网站（如 deepseek.com），点击页面右上角的“注册”按钮。填写用户名、邮箱地址和密码，完成基础注册信息提交。注意：如果遇到验证码接收失败，可尝试更换邮箱或检查垃圾邮件箱，或稍后重试。
⏱ 3分钟
📋 完成邮箱/手机验证
注册后系统会发送验证邮件或短信。登录邮箱，找到 DeepSeek 发送的验证链接并点击确认；若使用手机验证，输入收到的验证码完成绑定。验证成功后账号即激活。若未收到验证码，可在注册页面点击“重新发送”按钮，或检查手机号格式是否正确。
⏱ 4分钟
📋 登录并进入API管理页面
回到 DeepSeek 官网，使用注册的邮箱和密码登录。登录成功后，在用户头像或账户设置中找到“API 管理”或“开发者”选项，点击进入 API 密钥管理页面。若页面找不到入口，可参考官网文档或联系客服。
⏱ 3分钟
📋 创建并复制API密钥
在 API 管理页面，点击“创建新密钥”按钮，系统会生成一串唯一的 API 密钥。立即点击“复制”按钮将密钥保存到剪贴板，并粘贴到本地安全位置（如密码管理器或环境变量文件）。注意：密钥创建后不会再次完整显示，务必及时保存；若遗忘，需删除旧密
⏱ 5分钟
⚠️ 异常 DeepSeek 注册异常处理流程 ⏱ 15分钟
针对 DeepSeek 注册过程中可能遇到的验证码接收失败、密钥未保存、信息填写错误等常见问题，提供有序的排查与解决步骤。
⚠️ 核对注册信息
检查手机号或邮箱是否输入正确（如中国手机号需加+86），确认密码复杂度符合要求（至少8位含字母和数字）。若发现错误，点击返回上一步修改。若信息无误但验证码未收到，进入下一步。
⏱ 3分钟
⚠️ 重新获取验证码
在注册页面点击「重新发送验证码」按钮。若多次尝试仍无法接收，检查手机短信拦截/邮箱垃圾箱，或更换接收方式（如从手机号改为邮箱）。等待60秒后再次尝试。
⏱ 4分钟
⚠️ 处理密钥保存遗漏
若已成功创建账号但未及时保存 API 密钥，登录后进入 API 管理页面，在「密钥列表」中查找已存在的密钥并复制保存；若密钥未显示，可点击「创建新密钥」生成新密钥并立即备份。
⏱ 4分钟
⚠️ 最终验证与辅助方案
重新尝试完整注册流程，若仍失败，清除浏览器缓存/更换浏览器或网络环境（如切换4G/WiFi）。如异常持续，记录错误提示并联系 DeepSeek 技术支持（support@deepseek.com）或查看官方帮助文档。
⏱ 4分钟

场景 3 vc-s3-configure-zed ⏱ 15分钟
在 Zed 编辑器中为 Agent 配置 DeepSeek API 密钥，使其能够调用 AI 模型进行对话和代码辅助。
📋 打开 Zed Agent 面板
点击 Zed 编辑器左上角的星星图标（或使用快捷键 Ctrl+Shift+A / Cmd+Shift+A），打开 Zed Agent 侧边面板。
⏱ 2分钟
📋 进入模型选择与配置页面
在 Zed Agent 面板右下角找到倒数第二个按钮「Select a model」并点击，然后点击弹出的 Configure 按钮，进入模型配置页面。
⏱ 3分钟
📋 配置 DeepSeek API 密钥
在配置页面点击「LLM Provider」，从下拉列表中选择 DeepSeek。在密钥输入框中粘贴之前复制好的 DeepSeek API 密钥（如 sk-xxxxxxxx），确保无多余空格。
⏱ 5分钟
📋 验证配置并测试对话
关闭配置页面返回 Zed Agent 面板，在输入框中输入一条测试消息（例如“Hello”），点击发送。若正常收到回复，则配置成功。
⏱ 5分钟
